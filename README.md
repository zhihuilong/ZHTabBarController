## Requirements

Compatible with Xcode 8.2 / Swift 3.0

## Installation

ZHTabBarController is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "ZHTabBarController"
```
